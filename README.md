# AnimationsWeb

Proyecto Angular 21 con animaciones interactivas usando **D3.js** y estilos con **TailwindCSS**.

## Stack

- [Angular 21](https://angular.dev) — framework
- [D3.js](https://d3js.org) — gráficos y animaciones basadas en datos
- [TailwindCSS v4](https://tailwindcss.com) — estilos utilitarios
- [Vitest](https://vitest.dev) — tests unitarios
- [pnpm](https://pnpm.io) — gestor de paquetes

## Comandos

| Comando | Descripción |
|---------|-------------|
| `pnpm install` | Instalar dependencias |
| `pnpm start` | Servidor de desarrollo (`http://localhost:4200/`) |
| `pnpm build` | Build de producción en `dist/` |
| `pnpm test` | Ejecutar tests unitarios |
| `pnpm ng generate component <nombre>` | Generar un nuevo componente |

## Rutas

- `/home` — página principal
- `/animations` — demostraciones de animaciones con D3.js

## Formato

El proyecto usa [Prettier](https://prettier.io) para formateo de código.
